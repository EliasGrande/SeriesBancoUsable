# SeriesBancoUsable

SeriesBlanco se ha pasado tres pueblos poniendo publicidad invisible por todos lados, este escriptillo la hace usable de nuevo.

Instalación
-----------

  1. Instalar [Greasemonkey] \(para [Firefox]\) o alguna alternativa, como [Scriptish] \(para [Firefox]\) o [Tampermonkey] \(para [Chrome]\).
  2. Pulsa este [enlace al script] y acepta la instalación.
  
[Firefox]:https://www.mozilla.org/firefox
[Greasemonkey]:https://addons.mozilla.org/firefox/addon/greasemonkey/
[Scriptish]:https://addons.mozilla.org/firefox/addon/scriptish/

[Chrome]:https://www.google.com/chrome/
[Tampermonkey]:https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

[enlace al script]:series-blanco-usable.user.js?raw=true&.user.js
